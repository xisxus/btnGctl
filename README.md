# btnGctl

A simple button control system for handling trigger-based actions efficiently.

## 🔗 Live Demo

👉 https://xisxus.github.io/btnGctl/

## 🚀 Features

* Button-triggered actions
* Lightweight and fast
* Easy to use and integrate

## ⚙️ Setup

```bash
git clone https://github.com/xisxus/btnGctl.git
cd btnGctl
```

Open `index.html` in your browser or run with a local server.
